html学习笔记(三)

Posted on 2018-04-29

input：输入

- text
- password
- submit
- radio
- checkbox
- button
tips:type="radio"的input为单选按钮，若是佛那个多个选择中只能选一个，则多个input中必须要有相同的“name”属性值
checkbox为多选按钮，但也需要多个相同的"name"属性值

select：下拉列表

selected值表示默认选中

textaarea:定义多行文本输入字段

button：定义可点击的按钮
<button type="button" onclick="alert("Hello World!")"> Cick me!</button>
